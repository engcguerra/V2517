Controlo de Versões GIT

instalar o git https://git-scm.com/

verificar a versão do git git --version

começar um repositório local git init

podemos adicionar um ficheiro ao "staging area/ficheiros prontos para a próxima versão do programa"
git add nomeficheiro
git add .

para efetuar um commit / gravar uma versão do código git commit -m "mensagem com informação útil sobre o estado da app"

pode ser necessário configurar o autor e seu email git config --global user.email "you@example.com" git config --global user.name "Your Name"

consultar as várias versões da aplicação / vários commits git log
